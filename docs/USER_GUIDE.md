# 「皖美」小程序使用指南

## 功能介绍

### 1. 首页
- 查看各模块入口（美食、风景、文化、经济）
- 浏览每个分类的精选内容
- 快速导航到感兴趣的分类

### 2. 分类浏览
- **美食文化**：了解徽州毛豆腐、臭鳜鱼、黄山烧饼等特色美食
- **自然风景**：欣赏黄山、宏村、九华山、天柱山等美景
- **历史文化**：探索徽文化、黄梅戏、宣纸、徽墨等文化瑰宝
- **经济发展**：见证合肥科学中心、新能源汽车等发展成就

### 3. 内容详情
- 查看完整的内容介绍
- 点击底部图标进行点赞、收藏
- 发表评论与其他用户交流
- 分享精彩内容到微信好友或朋友圈

### 4. 我的页面
- 查看个人信息和收藏数量
- 修改昵称（点击昵称旁的编辑图标）
- 浏览所有收藏的内容

## 操作说明

### 点赞
1. 进入内容详情页
2. 点击底部的点赞图标
3. 再次点击可取消点赞

### 收藏
1. 进入内容详情页
2. 点击底部的收藏图标
3. 在"我的"页面查看所有收藏
4. 再次点击可取消收藏

### 评论
1. 进入内容详情页
2. 在评论输入框输入内容
3. 点击"发布评论"按钮
4. 评论将显示在评论列表中

### 分享
1. 点击微信右上角的分享按钮
2. 选择分享到好友或朋友圈
3. 好友可通过分享链接直接访问内容

### 修改昵称
1. 进入"我的"页面
2. 点击昵称旁的编辑图标
3. 输入新昵称
4. 点击"保存"按钮

## 注意事项

- 本小程序无需注册登录，首次使用会自动生成用户标识
- 点赞、收藏、评论等数据保存在云端
- 昵称和用户标识保存在本地，清除缓存后需重新设置
- 所有内容均为真实的安徽介绍信息
