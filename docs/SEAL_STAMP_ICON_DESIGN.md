# 首页分类按钮徽派印章风格设计

## 设计概述

将首页的3个分类按钮改为**徽派印章风格图标**，每个图标都是一个印章，内部是手绘的篆刻图案，体现徽州传统文化的精髓。印章图标线条使用徽墨色，背景采用不同的淡雅色调，既突出安徽特色，又保持视觉统一。

---

## 一、徽派印章设计理念

### 印章文化的意义
- **文化符号**：印章是中国传统文化的重要载体，徽州篆刻艺术尤为著名
- **身份标识**：印章代表身份和信誉，用于分类导航恰如其分
- **艺术价值**：篆刻线条流畅，图案精美，具有很高的艺术价值
- **地域特色**：徽州是篆刻艺术的重要发源地之一，体现地域文化

### 设计特点
- ✅ **印章形式**：方形或圆形印章外框
- ✅ **篆刻图案**：手绘风格的线条图案
- ✅ **徽墨线条**：使用徽墨色勾勒图案
- ✅ **淡雅背景**：不同分类使用不同的淡雅背景色

## 三、按钮样式设计

### 尺寸规格
- **容器宽度**：80px
- **印章图标尺寸**：70x70px（比之前的64px更大，更醒目）
- **文字大小**：12px
- **图标与文字间距**：6px
- **按钮间距**：20px

### 布局结构
```
┌──────────┐
│          │
│ [印章]   │  70x70px
│          │
│  文字    │  12px
└──────────┘
   80px宽
```

### CSS实现

```scss
/* 徽派印章风格图标按钮 */
.anhui-icon-btn {
  width: 80px;
  display: flex;
  flex-direction: column;
  align-items: center;
  justify-content: center;
  transition: all 0.3s ease;
}

.anhui-icon-btn:active {
  transform: scale(1.05);
}

.anhui-icon-image {
  width: 70px;
  height: 70px;
  margin-bottom: 6px;
  transition: all 0.3s ease;
  border-radius: 4px;
}

/* 选中态 - 印章红色发光效果 */
.anhui-icon-btn-active .anhui-icon-image {
  filter: drop-shadow(0 3px 10px rgba(230, 57, 70, 0.5));
  transform: translateY(-2px);
}

.anhui-icon-text {
  font-size: 12px;
  font-weight: 500;
  transition: all 0.3s ease;
}
```

## 四、选中态设计

### 印章红色发光效果

#### 视觉效果
- **红色阴影**：drop-shadow(0 3px 10px rgba(230, 57, 70, 0.5))
- **向上浮动**：translateY(-2px)
- **文字变红**：徽派红色

#### 设计寓意
- **盖章效果**：模拟印章盖在纸上的效果
- **红色印泥**：传统印章使用红色印泥，选中态用红色发光模拟
- **浮动效果**：向上浮动2px，增强选中的视觉反馈

### 实现代码
```tsx
<View className={`anhui-icon-btn ${selectedCategory === cat.key ? 'anhui-icon-btn-active' : ''}`}>
  <Image src={cat.icon} mode="aspectFit" className="anhui-icon-image" />
  <Text className={`anhui-icon-text ${selectedCategory === cat.key ? 'text-hui-red' : 'text-huimo'}`}>
    {cat.name}
  </Text>
</View>
```
