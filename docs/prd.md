# 「皖美视界」安徽文旅微信小程序需求文档

## 1. 小程序概述

### 1.1 小程序名称
皖美视界

### 1.2 小程序描述
展示安徽美好形象的微信小程序，全面介绍安徽的美食文化、自然风景、历史文化底蕴和经济发展水平，打造沉浸式文旅浏览体验。\n
### 1.3 产品定位
面向全国文旅爱好者和潜在游客的安徽文旅宣传平台，通过沉浸式内容展示、互动问答和便捷管理功能，全方位呈现安徽文旅魅力，提升用户对安徽的认知度和向往度。\n
## 2. 核心功能流程

### 2.1 用户浏览流程
1. 进入首页：顶部沉浸式通栏Banner展示安徽地标性景观，引导用户下滑探索
2. 浏览分类导航：横向滑动查看「黄山风光」「徽派建筑」「徽菜美食」「非遗民俗」「皖北风情」等核心分类
3. 浏览推荐内容：查看核心推荐区、地域专题区和轻量推荐区的精选内容卡片
4. 点击内容卡片：进入详情页查看美食文化、自然风景、文化展示或经济发展相关内容
5. 互动操作：对内容进行点赞、收藏、评论或分享
6. 参与徽文化小问答：在历史文化模块后方点击「徽文化小问答」按钮，完成5道题目答题，根据正确率获得「徽文化达人」印章

### 2.2 内容管理流程
1. 进入内容管理：在「我的」页面点击「内容管理」按钮
2. 选择管理模块：从题库管理、景点管理、美食管理、文化管理中选择需要修改的模块
3. 进行修改操作：\n   - 题库管理：编辑、新增或删除徽文化小问答题目（填写题干、选项、答案、解析）\n   - 景点/美食/文化管理：编辑对应内容的名称、图文介绍
4. 保存并预览：修改后自动保存，返回对应页面即可看到最新内容

## 3. 功能模块

### 3.1 首页沉浸式布局

#### 3.1.1 顶部沉浸式通栏Banner
- 背景：高清徽派建筑/黄山云海渐变图，低饱和度叠加10%黑蒙版
- 内容：居中放置「皖美视界」主标题（徽派书法体、白色、加粗）+「一脚踏进徽州，一眼看尽江淮」副标题（白色小字、居中）\n- 交互：Banner底部添加「向下滑动探索」轻量动效箭头
- 占屏比例：屏幕高度40%\n
#### 3.1.2 横向滑动分类导航栏
- 布局：横向可滑动导航，包含「黄山风光」「徽派建筑」「徽菜美食」「非遗民俗」「皖北风情」5个核心分类
- 样式：每个分类为手绘徽派风格圆形图标+文字组合，选中态加徽派红圆环标识
- 高度：80px

#### 3.1.3 核心推荐区
- 布局：2列等宽卡片，每行2个，卡片间距10px，左右边距15px
- 卡片样式：圆角12px+轻微阴影，顶部3:2比例高清实景图（宏村、臭鳜鱼等），底部标题加粗+简介小字+右下角「查看详情」小箭头

#### 3.1.4 地域专题区
- 布局：左侧文字区（40%）+右侧3张轮播小图（60%）\n- 内容：文字区写「徽州古村专题」+简短介绍，轮播图用宏村、西递、呈坎实景图，点击跳转专题页

#### 3.1.5 底部轻量推荐区
- 布局：1行3个小卡片（「热门打卡地」「新晋网红点」「文旅攻略」）\n- 样式：方形圆角浅灰色背景，仅展示图标+文字，简洁轻量化

- 点赞功能：用户可对内容进行点赞
- 收藏功能：用户可收藏喜欢的内容
- 评论功能：用户可对内容发表评论
- 分享功能：支持分享到朋友圈和好友

### 3.7 徽文化小问答模块

#### 3.7.1 功能入口与展示
1. 入口：历史文化按钮后面，新增「徽文化小问答」按钮，点进去后，模块背景用「淡徽墨灰+宣纸纹理」，标题「徽派文化小问答」用徽派书法体（深徽墨色），右侧加「开始答题」按钮（徽派红圆角按钮）\n2. 答题弹窗：点击「开始答题」后弹出全屏答题弹窗，弹窗背景为浅米黄宣纸纹理，顶部加徽派回纹装饰线，底部显示「剩余题目：X/5」\n
#### 3.7.2 答题核心逻辑
1. 题库配置（贴合安徽文旅）：\n   - 题目数量：默认5道题，涵盖徽菜、徽派建筑、徽州民俗（示例题目：\n     ① 马头墙最初的主要作用是？A.美观 B.防火 C.防盗
     ② 臭鳜鱼的传统腌制时间约为？A.1天 B.7天 C.15天
     ③ 黄梅戏的发源地是安徽哪个城市？A.安庆 B.黄山 C.合肥）\n   - 选项样式：每个选项为「徽派圆角卡片」（圆角8px+浅灰边框），选中时背景变为徽派红、文字变白，未选中为浅米黄

2. 答题交互：\n   - 答题流程：答完一题自动跳转到下一题，无需手动点击「下一题」\n   - 即时反馈：选择答案后，立刻提示「正确/错误」（正确用徽派绿+对勾图标，错误用徽派红+叉号图标），并显示简短解析（比如「马头墙又称防火墙，是徽派建筑的防火设计」）\n   - 完成奖励：答完5题后，弹出「答题完成」弹窗，显示正确率，并奖励1枚「徽文化达人」印章（自动存入个人中心的「印章集」）\n
#### 3.7.3 风格与兼容
1. 视觉风格：所有元素（按钮、弹窗、选项）的配色、字体、圆角，与小程序现有徽派风格统一（徽墨灰、浅米黄、徽派红）\n2. 兼容性：确保在微信小程序中弹窗弹出/关闭流畅，答题逻辑无卡顿，适配不同手机尺寸
3. 数据暂存：当前先将题库写死在前端（后续对接后端时替换为接口获取），答题记录暂存到小程序本地缓存（wx.setStorage）\n
### 3.8 内容管理功能

#### 3.8.1 修改入口
入口位置：在小程序“我的”页面，新增“内容管理”按钮（徽派红圆角样式，字体加粗，位置在“我的收藏”下方，显眼易找）\n
#### 3.8.2 小程序内内容修改页面

1. 内容管理总页面
   - 点击“内容管理”进入总页面，分4个模块：「题库管理」「景点管理」「美食管理」「文化管理」\n
2. 各模块修改逻辑

✅ 题库管理：\n   - 页面显示所有徽派问答题目，每道题右侧有“编辑”“删除”按钮，页面顶部有“新增题目”按钮
   - 编辑：点击“编辑”弹出表单（输入框+选项框），填写题干、选项A/B/C、答案、解析，点击“保存”即生效
   - 新增：点击“新增题目”，填完表单后提交，自动加入题库
   - 删除：点击“删除”，弹窗确认后即可删除题目

✅ 景点/美食/文化管理：\n   - 显示所有景点/美食/文化列表，点击条目进入编辑页（可修改名称、图文介绍），表单式填写，保存即生效

3. 操作反馈：\n   - 保存/删除/新增后，弹出“修改成功”“新增成功”的徽派风格提示弹窗
   - 修改后返回对应页面，能实时看到修改后的内容（比如改完题库后，去答题页面就能抽到改后的题目）\n
#### 3.8.3 核心适配要求
1. 无需任何权限验证（无密码、无身份区分）\n2. 所有修改操作仅在前端完成表单提交，数据同步到微信云数据库，修改后小程序内所有页面（答题、景点详情等）自动拉取最新内容
3. 操作流程极简：全程点击、输入、保存，无复杂步骤

## 4. 设计风格

### 4.1 视觉规范
- 配色：主色徽派红（#E63946）、皖蓝（#1D3557）、徽墨灰（#F1FAEE），全程低饱和度
- 间距：全局统一边距15px，模块间距20px，文字行高1.5倍
- 字体：标题徽派书法体，正文思源黑体，无衬线字体保证可读性
- 动效：分类栏滑动、专题区轮播、卡片点击轻微缩放3个轻量动效

### 4.2 整体布局
采用沉浸式文旅浏览逻辑，从顶部通栏Banner到底部轻量推荐区，营造连贯的视觉层次和浏览体验
