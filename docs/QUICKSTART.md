## 如何使用

### 1. 查看项目
```bash
cd /workspace/app-8f89mleb1ts1
```

### 2. 安装依赖（已安装）
```bash
pnpm install
```

### 3. 代码检查
```bash
pnpm run lint
```

### 4. 开发运行
```bash
# 微信小程序
pnpm run dev:weapp

# H5
pnpm run dev:h5
```

### 5. 构建
```bash
# 微信小程序
pnpm run build:weapp

# H5
pnpm run build:h5
```
