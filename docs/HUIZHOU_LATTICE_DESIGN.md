# 徽州窗棂纹理设计说明

## 设计理念

内容管理页面顶部采用「浅徽墨灰+淡徽州窗棂纹理」的设计，体现徽派建筑的传统美学。

## 视觉元素

### 1. 背景色
- **基础色**：浅徽墨灰（#F1FAEE）
- **色彩来源**：徽派建筑的白墙灰瓦

### 2. 窗棂纹理
徽州窗棂纹理由三层渐变线条组成，模拟传统徽派建筑的窗格图案：

#### 横向窗棂
- 间距：35px
- 线宽：2px
- 颜色：浅徽墨色，透明度8%
- 方向：水平（0度）

#### 竖向窗棂
- 间距：35px
- 线宽：2px
- 颜色：浅徽墨色，透明度8%
- 方向：垂直（90度）

#### 对角线纹理
- 间距：70px
- 线宽：1px
- 颜色：浅徽墨色，透明度3%
- 方向：对角（45度）

### 3. 文字配色
- **主标题**：徽墨色（#333333）- 深色，醒目
- **副标题**：浅徽墨色（#737373）- 中灰色，柔和
- **返回按钮**：徽墨色（#333333）- 与主标题一致

## 技术实现

### CSS类名
```css
.bg-huizhou-lattice
```

### 实现方式
使用CSS的`repeating-linear-gradient`叠加三层纹理：
1. 横向窗棂（0deg）
2. 竖向窗棂（90deg）
3. 对角线纹理（45deg）

### 代码示例
```tsx
<View className="bg-huizhou-lattice px-6 pt-8 pb-12">
  <View className="i-mdi-cog text-5xl text-huimo mb-3" />
  <Text className="text-2xl font-bold text-huimo mb-2">内容管理</Text>
  <Text className="text-sm text-huimo-light">全员可编辑 · 实时生效</Text>
</View>
```

## 设计特点

### 1. 文化传承
- 窗棂纹理源自徽派建筑的传统窗格设计
- 体现安徽地域文化特色
- 与小程序「皖美」主题高度契合

### 2. 视觉层次
- 淡雅的纹理不会干扰内容阅读
- 深色文字在浅色背景上对比清晰
- 三层纹理叠加形成丰富的视觉层次

### 3. 现代简约
- 纹理透明度控制在3%-8%，保持简洁
- 规则的几何图案符合现代审美
- 与卡片式布局完美融合

## 应用场景

目前应用于：
- 内容管理总页面顶部区域

可扩展至：
- 其他管理页面的顶部
- 特殊内容区域的背景
- 徽派主题的装饰元素

## 设计原则

1. **文化性**：体现徽派建筑美学
2. **可读性**：确保文字清晰可读
3. **简洁性**：纹理淡雅不喧宾夺主
4. **一致性**：与整体设计风格统一
