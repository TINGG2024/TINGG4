# 内容管理功能测试清单

## 测试环境
- 微信开发者工具
- 小程序预览模式

## 测试步骤

### 1. 入口测试
- [ ] 打开小程序，进入"我的"页面
- [ ] 确认"我的收藏"下方有"内容管理"按钮
- [ ] 确认按钮样式为徽派红色，字体加粗
- [ ] 点击"内容管理"按钮，能正常跳转

### 2. 内容管理总页面测试
- [ ] 确认显示4个模块：题库管理、景点管理、美食管理、文化管理
- [ ] 确认每个模块有图标、标题、描述
- [ ] 点击每个模块，能正常跳转到对应页面

### 3. 题库管理测试

#### 3.1 题目列表
- [ ] 确认显示所有题目
- [ ] 确认每道题显示题干、选项、正确答案、解析
- [ ] 确认每道题有"编辑"和"删除"按钮
- [ ] 确认顶部有"新增题目"按钮

#### 3.2 新增题目
- [ ] 点击"新增题目"按钮
- [ ] 填写题干、选项A/B/C、选择正确答案、填写解析
- [ ] 点击"保存"按钮
- [ ] 确认提示"新增成功"
- [ ] 返回列表页，确认新题目已添加

#### 3.3 编辑题目
- [ ] 点击某道题的"编辑"按钮
- [ ] 修改题干或选项
- [ ] 点击"保存"按钮
- [ ] 确认提示"修改成功"
- [ ] 返回列表页，确认题目已更新

#### 3.4 删除题目
- [ ] 点击某道题的"删除"按钮
- [ ] 确认弹出确认对话框
- [ ] 点击"删除"确认
- [ ] 确认提示"删除成功"
- [ ] 确认题目已从列表中移除

### 4. 景点管理测试
- [ ] 进入景点管理页面
- [ ] 确认显示所有景点列表
- [ ] 点击某个景点条目
- [ ] 修改标题、副标题、描述或图片URL
- [ ] 点击"保存"按钮
- [ ] 确认提示"修改成功"
- [ ] 返回列表页，确认内容已更新

- [ ] 修改标题、副标题、描述或图片URL
- [ ] 点击"保存"按钮
- [ ] 确认提示"修改成功"
- [ ] 返回列表页，确认内容已更新

### 6. 文化管理测试

- [ ] 确认提示"修改成功"

### 7. 数据实时生效测试

#### 7.1 题库实时生效
- [ ] 在题库管理中新增或修改题目
- [ ] 返回首页，进入"徽派问答"
- [ ] 开始答题，确认能看到新增或修改的题目

#### 7.2 内容实时生效
- [ ] 在景点/美食/文化管理中修改内容
- [ ] 返回对应的列表页（美食/风景/文化）
- [ ] 确认列表页显示最新内容
- [ ] 点击进入详情页，确认详情页显示最新内容

### 8. 表单验证测试
